# 云之星DIY教程：新手磁盘分区指南 (C盘分出D盘)

## 简介
很多品牌机或刚重装完系统的电脑，打开“此电脑”往往只看到一个 **C 盘**。
虽然现在的固态硬盘不强制分区，但为了**数据安全**（重装系统通常只格式化 C 盘）和**文件管理**，我们建议至少分出一个 **D 盘** 用来存放文档、软件安装包和个人资料。

本教程将教你利用 Windows 自带工具，安全、快速地“无损”分区，无需下载第三方软件。

## ⚠️ 操作前必读 (重要)

1.  **数据备份**：虽然此操作理论上无损，但涉及硬盘底层修改，**强烈建议**先备份重要数据，以防万一（如操作过程中突然断电）。
2.  **不要动隐藏分区**：在磁盘管理里，你会看到很多几百 MB 的小分区（如“恢复分区”、“EFI 系统分区”），**绝对不要删除或修改它们**，否则系统会无法启动！

## 第一步：打开磁盘管理工具

这是 Windows 掌管所有硬盘的后台。

1.  将鼠标移动到屏幕底部的 **Windows 徽标**（开始按钮）上。
2.  点击鼠标 **右键**（注意是右键）。
3.  在弹出的菜单列表中，选择 **“磁盘管理”** (Disk Management)。

*打开后，你会看到类似长条状的图表，我们要操作通常是最大的那个 **(C:)**。*

## 第二步：压缩 C 盘 (腾出空间)

我们需要先让 C 盘“瘦身”，把多余的空间挤出来给新盘。

1.  找到 **(C:)** 所在的那个长方形区域。
2.  鼠标 **右键** 点击它，选择 **“压缩卷”**。
3.  系统会提示“正在查询卷以获取可用的压缩空间”，请耐心等待（可能需要几十秒）。

## 第三步：计算并输入容量 (关键步骤)

弹出的窗口中，最关键的是 **“输入压缩空间量(MB)”** 这一栏。这里填写的数字，就是你未来 **D 盘的大小**。

### 🧮 容量计算公式
Windows 系统的换算规则是 **1 GB = 1024 MB**。
为了让你的 D 盘在“此电脑”里显示出整数（比如完美的 100 GB），不能直接填 100000。

**常用整数容量对照表：**

| 想要 D 盘大小 | 你需要输入的数值 (MB) |
| :--- | :--- |
| **50 GB** | `51200` |
| **100 GB** | `102400` |
| **150 GB** | `153600` |
| **200 GB** | `204800` |
| **300 GB** | `307200` |
| **500 GB** | `512000` |

1.  在“输入压缩空间量”里填入你算好的数值（例如 `102400`）。
2.  点击底部的 **“压缩”** 按钮。

*注意：压缩后 C 盘剩下的空间建议至少保留 60GB - 100GB，不要压榨得太狠，以免影响系统运行。*

## 第四步：新建 D 盘

压缩完成后，你会发现 C 盘右边多出了一块黑色的区域，上面写着 **“未分配”**。这部分就是刚才挤出来的空间。

1.  鼠标 **右键** 点击这块 **“未分配”** 的黑色区域。
2.  选择 **“新建简单卷”**。
3.  **新建简单卷向导** 弹出，操作如下：
    * **第一页**：点击“下一步”。
    * **简单卷大小**：默认是最大值，不用改，点击“下一步”。
    * **分配驱动器号**：默认应该是 **D**，如果被占用了可能会显示 E，通常不用改，点击“下一步”。
    * **格式化分区**：
        * 文件系统：保持 **NTFS**。
        * 分配单元大小：保持 **默认值**。
        * 卷标：可以默认为“新加卷”，也可以改成你喜欢的名字（如“软件”、“资料”）。
        * 点击“下一步”。
    * **完成**：点击“完成”。

## 第五步：大功告成

现在的磁盘管理界面中，原本黑色的“未分配”区域变成了蓝色的 **(D:)**。

打开“此电脑”，你就会看到全新的 D 盘已经准备就绪了！

## ❓ 常见问题 (Q&A)

**Q: 我后悔了，能把 D 盘合并回 C 盘吗？**
A: 可以，但步骤相反且有风险。
1. **必须先转移 D 盘所有数据**（合并会删除 D 盘）。
2. 在磁盘管理中右键 D 盘 -> **“删除卷”**（变回黑色“未分配”）。
3. 右键 C 盘 -> **“扩展卷”**，一直下一步即可吃掉后面的未分配空间。

**Q: 为什么我只能压缩一点点空间，明明 C 盘还很空？**
A: 这通常是因为有“不可移动的文件”（如系统页面文件）卡在磁盘中间。
* *解决方法*：暂时关闭系统虚拟内存和系统休眠，重启后再尝试压缩。如果太麻烦，可以使用第三方工具如 *DiskGenius* 进行无损分区（适合进阶用户）。

**Q: 只有一块硬盘，分 C 盘 D 盘会变慢吗？**
A: 在机械硬盘时代，分区过多可能会轻微影响寻道速度。但在现在的 **SSD (固态硬盘)** 上，分区对速度的影响微乎其微，完全可以忽略不计。
