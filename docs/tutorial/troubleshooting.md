## 🛠️ 第一阶段：内存检测 (排查蓝屏/闪退)
**使用选项：** `[1] TM5 内存测试`

内存频率不稳定（尤其是开启 XMP/EXPO 后）是电脑蓝屏最常见的原因。
* **操作：** 启动测试后，观察弹出的窗口。建议运行 **20分钟**。
* **判定：** * **正常：** 窗口下方 `Error` 计数始终为 **0**。
    * **故障：** 只要出现红色数字报错，说明内存不稳定。
* **对策：** 请重启进入 BIOS，**关闭 XMP/EXPO 选项**后再进入系统测试。若不再报错，说明是内存频率过高导致的不稳。

---

## 🎮 第二阶段：显存检测 (排查花屏/黑屏)
**使用选项：** `[2] FurMark 显存测试`

显存损坏通常会导致游戏花屏、屏幕出现异色点，或在负载升高时直接黑屏。
* **操作：** 脚本会启动显存专项测试，请根据您的显存容量选择对应数字。
* **判定：** * **正常：** 画面流畅，无杂色斑点，未跳出错误弹窗。
    * **故障：** 测试中途出现 **驱动重置、屏幕闪烁、花屏或报错**，基本判定为显卡硬件故障。

---

## ⚡ 第三阶段：CPU 核心检测 (排查死机/瞬间重启)
**使用选项：** `[3] Prime95 CPU测试` 或 `[4] AIDA64 FPU 烤机`

CPU 体质不足或散热问题会导致电脑在重负载下直接死机或保护性重启。
* **操作：** 推荐运行选项 `[3]`，这是业内公认最严苛的压力测试。建议观察 **15分钟**。
* **判定：** * **正常：** 电脑运行平稳，无掉线程或自动关机现象。
    * **故障：** 运行瞬间 **自动重启** 或弹出 **蓝屏**，说明 CPU 电压不稳或散热系统已达上限（温度瞬间过高）。

---

## 🔥 第四阶段：全负载双烤 (排查电源/整机稳定性)
**使用选项：** `[5]` 或 `[6]`

如果前面单项测试都没事，但玩大型游戏就重启，通常说明是 **电源 (PSU) 功率不足**。
* **操作：** 脚本会同时拉满 CPU 和显卡的功耗。启动后，请在菜单中根据您的显存选择测试档位。
* **判定：** * **直接断电：** 如果双烤刚开启不久电脑就 **瞬间黑屏断电**，说明电源无法承载整机瞬间功耗，需要更换更高功率的电源。
    * **一切正常：** 若能坚持 30 分钟不关机，说明硬件基本无隐患，故障可能由系统分区坏道或驱动冲突引起。

## 📝 故障总结参考表

| 故障现象 | 疑似元凶 | 核心验证工具 |
| :--- | :--- | :--- |
| **蓝屏 (Blue Screen)** | 内存稳定性 | `[1] TM5 测试` |
| **花屏 / 闪烁 / 掉驱动** | 显卡硬件 | `[2] FurMark 显存测试` |
| **瞬时断电重启** | 电源功率不足 | `[5]` 或 `[6]` 双烤测试 |
| **画面卡死 / 死机** | CPU 或 散热 | `[3] Prime95 测试` |

> **💡 温馨提示：** 测试期间硬件会产生大量热量，风扇转速提高属于正常现象。若由于散热不佳导致自动关机，请及时清理灰尘或更换硅脂。
