# 故障排查

## 快速判断（建议先看）

- 是否能开机（风扇/灯效/主板指示灯）
- 是否有画面输出
- 是否能进入系统

## 场景 1：开机无显示（占位）

- 检查显示器信号源与线材
- 检查显卡供电/重新插拔显卡
- 检查内存条（单条轮换测试）

## 场景 2：频繁蓝屏/死机（占位）

- 关闭 XMP/EXPO 排查内存稳定性
- 查看温度与供电
- 更新 BIOS 与驱动

## 场景 3：温度高/噪音大（占位）

- 检查风扇方向与转速曲线
- 检查硅脂与散热器安装压力

## 需要人工协助时请准备的信息

- 配置清单
- 故障现象描述与复现条件
- 蓝屏代码/事件查看器日志（占位）
