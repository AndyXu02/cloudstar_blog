# 重做系统

## 适用范围

- 电脑系统异常、需要清理重装、或更换硬盘后

## 准备工作

- 备份重要数据（桌面、文档、浏览器书签等）
- 准备 U 盘（建议 8GB 及以上）
- 记录网卡驱动/主板型号（占位）

## 重装流程（占位）

### 1. 制作启动盘

- 使用官方工具/镜像制作

### 2. BIOS 设置启动项

- 进入 BIOS
- 选择 U 盘启动

### 3. 分区与安装

- 删除/新建分区（按需）
- 安装系统

### 4. 驱动与基础软件

- 芯片组/网卡/显卡驱动
- Windows 更新

## 常见问题

- 找不到硬盘/需要加载存储驱动（占位）
- 安装后无网络（占位）
